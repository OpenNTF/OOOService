Domino OOO

Domino Out Of Office Service

This is the location of the Eclipse update site for the Domino out-of-office Rest service

Open the update site

Installation Option 1

Copy the plugins from this site (see the XML description for the latest) to [DominoAppDir]/osgi/shared/eclipse/plugins

Installation Option 2

Import this update site to an updateside.nsf and use that in Domino

Sample JSON

{
"user": "jane.doe@projectcastle.io",
"enabled": true,
"out": "2017-03-09T10:00:00.000-0600",
"in": "2017-03-15T11:00:00.000-0500",
"subject": "I am out of the office from Fri 03/10/2017 until Thu 03/16/2017.",
"body": "I'm gone fishing!",
"lastUpdate": "2017-03-13T02:22:44.413-0500"
}

If there is any error condition, the resulting JSON object will have an error element.

Use

Call the URL https://your.dominoserver.com/oooservice/q/some.email@yourorg.com to retrieve the user's status

Parameters

  • debug=true to get additional JSON output
  • force=true to bypass the 30 min cache

Statistics

The Domino console will provide show stat OOO (that's the letter O, not number 0) for performance statistics. Alternative the URL https://your.dominoserver.com/oooservice/stats provides the same info
